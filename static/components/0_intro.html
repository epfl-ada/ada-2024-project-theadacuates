Main Question:
Is Cognitive Fatigue Quantifiable in Wikispeedia, and does it Impact the Gameplay?

Recent studies show that our attention spans are shrinking, and our brains can become fatigued within minutes of focused activity (Ref). This project aims to investigate user engagement with the Wikispeedia game from a neuroscientific angle, focusing on how article complexity impacts cognitive load and behavior. Key questions include:

  • How does network structure influence player navigation?
  • What role do article content, layout and complexity play in path completion?
  • Are there biological biases that influence the gameplay?

The dataset used in this analysis can be found here. It contains the paths from approximately 75,000 games, along with additional information such as the game’s rating and the number of back-clicks. The primary focus of the dataset is the paths, i.e. sequences of articles that users followed to navigate from the source to the target link.

Before proceeding with the data analysis, it is important to clarify some aspects and hypotheses on which the analysis is based:

  • The division between finished and unfinished paths will be used as a proxy for mental fatigue, hypothesizing that unfinished paths occur due to some brain bias.
  • The motivation of the player, given that Wikipedia is a non-lucrative game, will be assumed to be the same for all participants. Therefore, motivation will not be considered a confounding factor in the analysis. (See more on how to measure motivation neuroscientifically).
  • Each path has an associated IP address. However, due to the intrinsic nature of IP addresses (Ref), it is not possible to link each address to a unique individual. Consequently, no evolution of player behavior is studied. Instead, each path is assumed to be an independent sample, i.e., played by a different person, for statistical analysis.

Meet Our Characters:

Pepito is a young neuroscience student with big dreams and... questionable study habits. Last week, he faced an unexpected academic crisis—not in the lab, but in a game called Wikispeedia. The result? Pepito got lost. A lot. Like, "How did I end up reading about Cabbage Moths?" lost.
Embarrassed but determined, Pepito boldly blamed the game, not himself. "It’s not me—it’s cognitive overload! This game would exhaust even the sharpest minds."
 But Pepito's mentor, the legendary Dr. Ramón y Cajal, wasn’t buying it. Known for spotting an excuse from a mile away, Dr. Cajal challenged Pepito to prove his claims with data.
What follows is a heated back-and-forth between Pepito and Dr. Cajal. Could Pepito’s data reveal the game’s inherent challenge, or would Dr. Cajal’s “study harder” mantra prevail?

Wikispeedia

Wikispeedia is a human-computation game in which users are asked to navigate from a given source to a given target article by only clicking Wikipedia links.
